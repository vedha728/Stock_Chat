ABSTRACT

The Indian retail investment landscape has seen rapid growth in recent years, with millions of new investors entering the stock market through digital platforms. However, most retail investors lack the technical expertise to interpret complex chart patterns, track global macroeconomic shifts, or analyze news sentiment — all of which significantly influence stock price movements. Existing platforms such as TradingView and Tickertape provide isolated tools for either technical charting or fundamental screening, but none offer an integrated, machine-learning-driven advisory system that fuses multiple signal domains into a single actionable recommendation.

This project presents a Multi-Signal Stock Advisory System that addresses this gap by combining three distinct analytical signal domains — Technical Chart Indicators, News Sentiment Analysis, and Global Macro Indicators — into a unified XGBoost-based classification model. The system fetches real-time market data from Yahoo Finance, processes live news headlines through Google Gemini LLM for sentiment scoring, and incorporates S&P 500, Crude Oil, and USD/INR exchange rate returns as macroeconomic context. A 19-feature vector is constructed and evaluated against a pre-trained XGBoost model to produce a BUY, HOLD, or SELL verdict with a confidence percentage. The system also features a 10-day historical signal trend tracker, fundamental ratio displays, and a strategy backtesting module. The entire application is deployed as a serverless web application on Vercel, with a React-based frontend and Python-based API backend, making it freely accessible to any retail investor.

Keywords: Stock Market Prediction, XGBoost, Sentiment Analysis, Technical Indicators, Multi-Signal Fusion, Indian Stock Market, NSE, Machine Learning, Serverless Deployment.

CHAPTER 1
INTRODUCTION

1.1 Problem Statement

The Indian stock market has witnessed an unprecedented surge in retail investor participation, with the number of demat accounts crossing 15 crore in recent years. Despite this growth, a significant majority of retail investors make investment decisions based on social media tips, unverified Telegram channels, or impulsive reactions to market volatility. Professional-grade advisory services are typically expensive, often costing thousands of rupees per month, placing them beyond the reach of the average investor.

Existing free platforms offer fragmented tools — charting platforms display technical indicators without interpretation, news portals report headlines without quantifying sentiment impact, and fundamental screening tools provide static ratios without directional advice. No widely available free platform combines these distinct analytical dimensions into a single, machine-learning-driven advisory verdict tailored to the Indian market.

This gap exposes retail investors to uninformed decision-making, where critical signals such as bearish momentum divergence, negative news sentiment, or adverse global macroeconomic conditions are overlooked, leading to poorly timed entries and exits.

1.2 Objective

The objective of this project is to design and develop a web-based Multi-Signal Stock Advisory System that provides real-time BUY, HOLD, or SELL recommendations for stocks listed on the National Stock Exchange (NSE) of India. The system aims to:

  • Integrate multiple analytical signal domains — technical chart indicators, news sentiment analysis, and global macroeconomic indicators — into a unified machine learning model.
  • Employ an XGBoost gradient-boosted classification model trained on a 19-feature vector derived from historical price data, curated news sentiment, and global market correlations.
  • Provide real-time advisory verdicts with confidence percentages, enabling investors to make informed decisions without requiring deep technical expertise.
  • Deploy the system as a freely accessible serverless web application, ensuring zero installation requirements for end users.

1.3 Scope

The system currently supports over 110 actively traded large-cap and mid-cap stocks on the NSE. The coverage includes major indices constituents across sectors such as Information Technology (TCS, Infosys, Wipro), Banking (HDFC Bank, ICICI Bank, SBI), Automobiles (Tata Motors, Maruti Suzuki), Pharmaceuticals (Sun Pharma, Dr. Reddy's), and Energy (Reliance Industries, ONGC), among others.

The system provides the following functional capabilities: (a) a real-time multi-signal analysis dashboard with verdict card, probability distribution, and signal summaries; (b) a 10-day historical signal trend tracker that visualizes how the advisory verdict has changed over the past two trading weeks; (c) fundamental ratio displays including P/E ratio, P/B ratio, Debt-to-Equity ratio, and market capitalization; (d) a side-by-side stock comparison mode; and (e) a strategy backtesting module that simulates historical trading performance based on model signals.

2.3 Data Flow Workflow

When a user submits a stock ticker for analysis, the system executes the following workflow sequentially:

  1. Ticker Resolution: The user's query is matched against a supported stock dictionary to resolve the correct NSE ticker symbol (e.g., "Reliance" → "RELIANCE.NS").
  2. Concurrent Data Fetching: The API backend dispatches parallel requests using Python's ThreadPoolExecutor to fetch: (a) 1-year price data, (b) live headlines, (c) FII/DII net flow, and (d) global macro returns.
  3. Technical Indicator Computation: The system computes RSI (14-day), MACD histogram, MACD crossover, moving averages (50-day and 200-day), golden cross, and volume ratios.
  4. Feature Vector Construction: A 19-dimensional feature vector is compiled from technical indicators, Gemini sentiment scores, macro returns, and momentum window metrics.
  5. Model Evaluation: The feature vector is passed to a custom Python XGBoost JSON tree evaluator that walks tree nodes at runtime to compute softmax class probabilities.
  6. Verdict Generation: The class with the highest probability is selected as the verdict, subject to a 55% minimum confidence threshold (defaulting to HOLD if low confidence).
  7. Response Rendering: The API returns a structured JSON payload containing the verdict, probabilities, metrics, and advisory notes, which the React dashboard renders immediately.

3.2 XGBoost Classification

The advisory model is built using XGBoost (Extreme Gradient Boosting), a gradient-boosted decision tree ensemble algorithm. The model is trained as a multi-class classifier with three output classes: BUY (class 0), HOLD (class 1), and SELL (class 2). Training is performed on one year of historical data across 110 NSE stocks, with labels assigned based on forward-looking 5-day returns: if the price increased by more than 2%, the label is BUY; if it decreased by more than 2%, the label is SELL; otherwise, HOLD.

A critical engineering decision in this project was the implementation of a pure-Python XGBoost JSON tree evaluator. Serverless platforms such as Vercel impose strict binary dependency restrictions and memory limits that prevent the installation of the native XGBoost C++ library. To overcome this, the trained model is exported to a JSON tree structure, and a custom Python function walks through the decision trees at inference time — evaluating split conditions, traversing left/right child nodes, and aggregating leaf values to compute softmax class probabilities. This approach eliminates the need for any compiled binary dependencies while producing mathematically identical results to the native library.

A minimum confidence threshold of 55% is enforced: if no class probability exceeds this threshold, the system defaults to a HOLD verdict to avoid providing low-confidence directional advice.

3.3 10-Day Signal Trend Tracker

The system includes a 10-day historical signal trend tracker that visualizes how the advisory verdict has evolved over the past ten trading days. For each historical day, the system slices the price data up to that day only (preventing future data leakage), recomputes all 19 features using only information available on that date, evaluates the XGBoost model, and records the resulting verdict and confidence score. This produces a time-series trend line displayed as an interactive Recharts area chart on the dashboard, with a pulsating marker on the most recent day's data point.

By evaluating the 19 features independently for each of the past ten trading days, the tracker allows the user to see if a stock's bullish signal is strengthening (e.g., transitioning from a low-confidence HOLD to a high-confidence BUY) or weakening. This trend context prevents retail investors from executing trades based on temporary noise or minor, single-day spikes.

CHAPTER 5
MERITS AND DEMERITS

5.1 Merits

  1. Multi-Signal Fusion: Unlike platforms that offer isolated tools, this system fuses three independent signal domains (technical indicators, news sentiment, and global macro indicators) into a single machine-learning-driven verdict, providing a holistic view of stock conditions.
  2. Real-Time Analysis: The system fetches live market data, computes indicators, and generates predictions in real-time. Concurrent API fetching using thread pools ensures the entire analysis completes within Vercel's serverless timeout budget.
  3. Pure-Python Serverless Evaluator: A custom XGBoost JSON tree walker was implemented to bypass binary dependency restrictions on serverless platforms. This innovation enables ML model deployment on platforms where compiled C++ libraries cannot be installed.
  4. 10-Day Historical Trend Visualization: The system provides a unique trend tracker that shows how the advisory verdict has evolved over the past 10 trading days, helping investors understand momentum direction rather than relying on a single point-in-time recommendation.
  5. Context-Aware Advisory Notes: The system detects user intent from the search query (e.g., "should I buy TCS" vs. "should I sell TCS") and tailors the advisory explanation to match the user's perspective, improving the relevance and usability of the advice.
  6. Free and Accessible: The entire system is deployed on Vercel's free tier and requires no installation, registration, or subscription, making professional-grade stock analysis accessible to any retail investor with a web browser.

CHAPTER 5
MERITS AND DEMERITS

5.2 Demerits

  1. Limited Stock Coverage: The model is currently trained on approximately 110 large-cap and mid-cap NSE stocks. While the system can analyze any stock through dynamic ticker resolution, prediction accuracy may be reduced for micro-cap or newly listed stocks that were not part of the training dataset.
  2. FII/DII Data Not in Model: Foreign Institutional Investor (FII) and Domestic Institutional Investor (DII) flow data is fetched and displayed on the dashboard but is not yet integrated into the XGBoost model's feature vector. This data currently serves as a visual advisory aid only.
  3. Headline-Level Sentiment Analysis: The news sentiment engine analyzes headline text only, not full article bodies. This can occasionally lead to incomplete sentiment assessment when the headline tone differs from the article's detailed content.
  4. No Portfolio Tracking: The system provides advisory recommendations only and does not offer user authentication, portfolio management, or direct trade execution capabilities.
  5. Serverless Timeout Constraints: Vercel's free tier imposes a 60-second function execution limit. During periods of high API latency from upstream data providers (Yahoo Finance, Gemini), the analysis may occasionally time out.

COMPARISON WITH EXISTING WORK

This chapter compares the Multi-Signal Stock Advisory System with two widely used existing platforms to highlight the improvements and differences offered by this project.

6.1 TradingView

TradingView is a globally popular charting and technical analysis platform used by traders worldwide. A comparison with the proposed system reveals the following distinctions:

  • TradingView provides advanced interactive charting tools with a wide range of technical indicators such as RSI, MACD, Bollinger Bands, and Fibonacci retracements. However, the interpretation of these indicators is left entirely to the user — the platform does not generate automated BUY, HOLD, or SELL recommendations.
  • TradingView does not incorporate news sentiment analysis. Users must manually track financial news from external sources and assess its potential impact on stock prices.
  • Global macroeconomic indicators such as S&P 500 correlation, crude oil returns, and currency exchange impacts are not integrated into any automated advisory output.
  • Full access to TradingView's features requires a paid subscription (Essential, Plus, or Premium plans), placing comprehensive analysis behind a paywall.

Our Improvement: The proposed system automates chart interpretation through an XGBoost machine learning model, combines it with real-time sentiment and macro signals, and delivers a unified advisory verdict — all accessible for free without any subscription.

6.2 Tickertape

Tickertape is an Indian financial data platform focused on fundamental analysis and stock screening. The following differences are noted:

  • Tickertape excels at fundamental screening, providing detailed financial metrics including P/E ratio, P/B ratio, return on equity, and debt ratios. It also offers stock comparison tools and curated screeners for different investment styles.
  • Tickertape does not provide real-time machine-learning-based directional predictions. Its analysis is based on static financial ratios and does not produce actionable BUY or SELL recommendations.
  • The platform does not perform real-time news sentiment analysis or integrate live sentiment scores into its stock evaluation.
  • Tickertape does not offer historical signal trend tracking or strategy backtesting based on ML-generated signals.

Our Improvement: While Tickertape provides excellent static fundamental data, our system adds a dynamic, real-time predictive layer by fusing technical indicators, sentiment analysis, and macro signals into an ML model that provides directional advice with confidence scores and historical trend context.

6.3 Summary

Existing platforms such as TradingView and Tickertape provide valuable but isolated analytical tools — either technical charting or fundamental screening. The Multi-Signal Stock Advisory System differentiates itself by fusing three signal domains into a unified machine-learning model that generates real-time, actionable advisory verdicts. This end-to-end integration, combined with free accessibility and serverless deployment, represents a distinct contribution to the Indian retail investment ecosystem.

CHAPTER 7
CONCLUSION & FUTURE SCOPE

7.1 Conclusion

This project successfully designed, developed, and deployed a Multi-Signal Stock Advisory System that addresses the information asymmetry faced by retail investors in the Indian stock market. By combining three distinct analytical signal domains — technical chart indicators, news sentiment analysis, and global macroeconomic indicators — into a unified XGBoost-based classification model, the system generates real-time BUY, HOLD, or SELL recommendations with confidence percentages for over 110 NSE-listed stocks.

The system was deployed as a serverless web application on Vercel, featuring a React-based frontend with interactive data visualizations and a Python-based API backend. A key technical contribution of this project is the implementation of a pure-Python XGBoost JSON tree evaluator, which enables machine learning model inference on serverless platforms without requiring compiled binary dependencies. The 10-day signal trend tracker provides investors with historical context for the advisory verdict, moving beyond single-point predictions to trend-aware analysis.

The project demonstrates that a multi-signal fusion approach to stock advisory can be implemented as a practical, freely accessible tool for the Indian retail investment community, bridging the gap between sophisticated institutional analysis and individual investor capabilities.

7.2 Future Scope

The following enhancements are planned for future versions of the system:

  1. Integrate FII/DII Flows into the ML Model (5-Signal Upgrade): Foreign Institutional Investor and Domestic Institutional Investor net flow data is currently displayed on the dashboard as a visual advisory aid. In future iterations, this data will be incorporated directly into the XGBoost model's feature vector, enabling the model to learn correlations between institutional buying/selling patterns and subsequent price movements. This upgrade would transform the system from a 3-signal to a 5-signal advisory engine.
  2. Expand to All NSE/BSE Stocks: The current system supports approximately 110 curated stocks. By leveraging Yahoo Finance's dynamic ticker resolution API, the system can be expanded to support all actively traded stocks on both the NSE and BSE exchanges. This would involve replacing the static stock dictionary with a dynamic search-and-resolve mechanism that maps user queries to valid ticker symbols in real-time.
  3. Add User Portfolio Tracking and Personalized Watchlists: Future versions will introduce user authentication and portfolio management features, allowing investors to maintain a personal watchlist, track their holdings, receive automated alerts when the advisory verdict changes for a watched stock, and view portfolio-level risk assessments based on the model's multi-signal analysis.
